# SportIL Generated Facility Images

These bitmap assets are AI-generated representative visuals for the SportIL prototype.

They are not photographs of specific real facilities and must not be used as proof of facility appearance, equipment, accessibility, or current condition.

Current assets were generated from SportIL prompts using the Pollinations image endpoint with a Flux model seed per sport/facility category, then normalized to 1600x900 progressive JPEG.

Provider reference: https://pollinations.ai/ and https://gen.pollinations.ai/docs

Regenerate with:

```bash
python3 apps/web/scripts/generate-sportil-assets.py
```

Production replacement still needs rights-cleared real photography or a dedicated approved generative-media pipeline with legal review, accessibility alt text, and per-facility media provenance.
